Key/Button Action
Mouse wheel Zoom in/out to location under cursor.
Left click and drag Pan the view.
Left click On node: selecting the node. On background: de-select all.
Left double click Load children for the selected node, also complements missing children.
Right-Click and Drag Move current selection to drop target. Operation aborted if dropped on background.
Ctrl-Right-Click and Drag Create additional link from the currently selected node to drop target. Operation aborted if dropped on background.
Space Expanding or collapsing nodes
Right/Left Arrow Navigating hierarchy down/up from selection.
Up/Down Arrow Navigating to sibling. With ALT key pressed: swapping position with sibling.
Delete Options: Hide (removes the node from the map but does not alter the database), Hide + Unlink (removes the node and unlinks the two Mindliner objects permanently), Hide + Unlink + Delete (removes the node, and deletes the selected object permanently).
A Key Toggle anchor status for the current selection. Anchors only have a meaning for dynamic maps where they won't be moved.
CTRL-E Start editing session on selected node (toobar switches to editor).
Save New Map Saves the current layout on the server (currently all maps are visible to others). Note: It is not neccessary to save maps unless the layout is manually edited after map creation. If nodes are edited the changes are automatically stored for the base object.
Load Map Replaces the current map with the contents of the loaded map. Nodes for which the base object has been deleted in the meantime are no longer shown.
Update Map This options is available only after a map has been loaded. For newly built maps use Save New Map. The update map function updates all existing objects with the alterations and adds new nodes. (it does not delete nodes yet)
Expand Contacts Contact nodes can have very many children. If this checkbox is off then contacts are not automatically expanded and build performance is better for maps with contact nodes.
C Key Assign a new color to the selected node.
F Key Assign new font to the currently selected node.
R key Add all missing children of the selected node excluding the parent. This ensures that any newly added or previously hidden nodes are represented in the map.
