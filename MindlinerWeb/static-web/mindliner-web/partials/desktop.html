When Do I Need MindlinerDesktop?

If you use Mindliner daily and have enough computer resources to keep it running throughout the work day then you will get a significant general performance boost from the DesktopApp no matter what you are using it for. In addition there are the features described below which are only available in the DesktopApp.

MINDMAPPING

  • custom color schemes to highlight objects through attributes
  • upload and index files through drag & drop
  • linking one and the same object to multiple "parents" in the mind map
  • text formatting of all nodes while preserving display hierarchy
  • optional display of description and attributes on map node

CACHING & OFFLINE MODE

  • object and attribute cache speeds up search, presentation, and navigation
  • offline search, editing, and creation of objects with automatic re-synch when going online

EDITING

  • hanging attributes for multiple objects at once
  • specific contact editor including profile picture update

SEARCH LIST

  • choice of columns to display in results list
  • drag and drop an object (or objects) from the search list onto an existing node in the mind map to link it up

WEEKPLANS & TIME TRACKING

  • creation of weekplans using existing and new tasks
  • creation of week baseline plan
  • tracking of time against tasks on the plan with comparison to baseline plan
  • display of own plans together with those of the team

WORKSPHERE MAP

  • mapping workflows, responsibilities and processes for knowledge transfer or documentation
  • interactively and visually classifying objects
  • charts like the business model canvas
